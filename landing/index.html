Feature Highlights

Theory Engine

31 scales x 12 keys. Diatonic chords. Available Scale reverse lookup. Pivot chord analysis for modulation.

Voicing Tools

Drop 2 / Drop 3, Shell voicing (1-3-7, 1-7-3), inversions, Omit 5, Rootless. Voicing box with multiple fingering options.

Sound & Performance

Built-in organ and electric piano (Rhodes sampler). Effects. Perform mode for live chord progressions. No external sound source needed.

MIDI & Integration

WebMIDI input/output. MIDI file export. Push 3 octave sync. Velocity sensitivity controls. Works with any MIDI pad controller.

Okabe-Ito color palette (color-vision accessible). All data stays in your browser (localStorage). No tracking, no cloud.
