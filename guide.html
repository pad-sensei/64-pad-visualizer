64 Pad Explorer

64パッド上のスケール・コード・ボイシングを探索するブラウザツール。
インストール不要 — ブラウザだけで動きます。

アプリを開く

特徴

  • 無料 — すべての機能を無料で使えます
  • インストール不要 — ブラウザだけで動きます。スマートフォンでもPCでも。スマートフォンでは32パッドで演奏できます
  • 複雑なセットアップ不要 — Pushを繋ぐだけで使えます。MIDIの設定は必要ありません
  • パッドのどこを押せばいいか見える — コードやスケールを選ぶと、64パッド上に手形が色で表示されます。押さえ方がひと目でわかります
  • コードを様々な角度から調べられる — テキスト入力、ダイアトニックバー、コードビルダー、手形、ギター指板、鍵盤。入口がたくさんあります
  • コードを保存できる — 16パッドにコードを保存して、好きなコード進行を試すもよし、気に入ったコードを保存してMIDIファイルに書き出すこともできます
  • 音源内蔵 — Rhodes物理モデリング音源搭載。エフェクト付き。表示されたコードやスケールをその場で聴けます。ダウンロード不要で即座に発音します
  • Available Scaleをコードにオーバーレイ — コードに対して使えるスケールを重ね合わせて表示。DiatonicとPracticalの2つのソートで、機械的な分析だけでなく実践に即した分析ができます
  • ハイブリッドコード対応 — C/DやF△7/Gのような分数コードも実践的に分析できます
  • セクション折りたたみ — 必要な情報だけを確認できます
  • 3ペイン並び替え — Chord・Pad・Circleの表示順を6パターンから選択可能
  • 豊富なショートカットキー — ⌘⌥+L(Link) / G(Guitar) / B(Bass) / P(Piano) / F(五度圏) / S(五線譜) / A(Audio) / Q(Quality) / V(Voicing) / K(Key) / H(Header) / N(Harm/Mel) / D(SecDom) / R(Parallel)で自分に必要な情報を選択できます。覚えやすい工夫がされています
  • 多言語対応・色覚多様性対応 — 9言語（EN / JA / ZH / ES / FR / PT / DE / KO / IT）。色覚バリアフリー配色

これだけの機能がひとつのブラウザアプリに入っています。

画面紹介

Scale Scaleモード

キーとスケールを選択すると、64パッド上にスケール音が色分けで表示されます。ダイアトニックバーからコードを選択できます。

C Major (Ionian) — ルート音(オレンジ)、スケール音(ブルー)、特性音(黄)で色分け

Chord Chordモード

Root → Quality → Tension の3ステップでコードを構築。ボイシングの転回・Drop・Shell・Omitも操作できます。Parent Scaleパネルでそのコードが属しうるスケールを逆引きできます。

Dm7 — コードトーン、3rd(ピンク)、7th(グリーン)を表示。ボイシングボックスで配置候補を確認

Input Inputモード

MIDIコントローラー、パッドをクリック、指板、鍵盤などからコードを入力できます。コードの判定は基本的にどのモードでも動作します。

C7 — パッドをクリックするだけでコード名を自動判定。複数候補も表示

Performモード

メモリーに保存したコードを16パッドに配置し、キーボードやMIDIパッドでリアルタイム演奏できます。

Performモード — C7, Fm7, G7, Am7をスロットに保存。キーボード4×4グリッドで演奏

MIDI感度調整（Velocity Sensitivity）

MIDIコントローラーのベロシティ感度をPush 3風の4パラメータで調整できます。サウンドパネル内のVELOCITY SENSITIVITYセクションで設定。カーブがリアルタイムに可視化されます。感度調整がないMIDI機器に便利です。

  • THRESH (Threshold) — 最低入力ベロシティ。これ以下のタッチは無音になります（0〜64）
  • DRIVE — カーブの立ち上がり。+で弱いタッチでも大きな音に、-で強く叩かないと鳴らない（-64〜+64）
  • COMP (Compand) — ダイナミックレンジ圧縮(+)/拡張(-)。+で音量差が均一に、-で強弱の差が広がる（-64〜+64）
  • RANGE — 最大出力ベロシティ。127で最大、下げると全体的に弱くなる（1〜127）

Push 3だけでなく、LinnStrumentや他のMIDIコントローラーでも使えます。設定はlocalStorageに自動保存されます。

Push 3 オクターブボタン対応

Push 3のオクターブ▲▼ボタン（CC#55/CC#54）をアプリ内のオクターブシフトに連動させます。Push 3本体のオクターブボタンを押すと、アプリの表示オクターブが自動的に切り替わります。

Available Scale + ピボットコード

コードが属しうるスケールを逆引き表示。Practical（実用性順）/Diatonic（理論順）の切替が可能。手動コード入力時はダイアトニックバーが消え、ピボットコードとして分析できます。

Dm7 → C Major (Dorian), C Mel.Min (Dorian b2), F Major (Aeolian) 等。Available Tensionsも表示

分数コード（例: C/D, F△7/G）では、ベース音がコードトーン外の場合、ベース音をルートとしてAvailable Scaleを自動再計算します。ハイブリッドコードではミクソリディアンが第1候補になります。ベース音がコードトーン（転回形）の場合はルートを変更しません。

※ ペンタトニック/ブルーススケールはモード（横のアプローチ）であり、コード・スケール理論（縦の対応）とは別体系のため、Available Scaleでは扱いません。実戦ではペンタトニックが最初の選択肢になることが多いです。

HPS専用ボイシング機能

HPS会員は3種類のボイシング機能を使えます。TASTY / STOCKは選んだボイシングが自動で64パッドに表示され、ギターは「パッドに反映」で配置を切り替えられます。

TASTY Voicing — トップノートから分類

129種類の洗練されたボイシングを度数配列で表示。ボイシングがはじめから固定されて表示されるので、自分で組む必要がありません。コードを選んで T キーで起動。Z/Xでレシピを送ります。パッドに全音が収まる最適ポジションを自動選択します。コードのボキャブラリーを強化、ホーンアレンジなどにも役立ちます。

対象: Maj7 / Dom7 / Min7

info欄に度数の積み順（例: m9 (b3-b7-9-5-1)）が表示されます。これがボイシングの構造——下からどの順番で音を積むかを示しています。

ラベルの意味
  • Rootless — ルート（1度）を省略したボイシング。バンドではベーシストがルートを弾くため、鍵盤やパッドではルートを省いて上部構造だけを弾くのが一般的です。
  • Omit5 — 5度を省略。
  • Omit3 — 3度を省略。
操作
  • T — TASTY ON/OFF
  • Z / X — 前/次のレシピへ（TASTY/STOCK共通）
  • ←→ — ボイシング全体を半音トランスポーズ（フィンガリング形状を維持）
  • Escape — 元のコードに戻る
  • Space — 現在のボイシングを再生
  • Opt+キー — メモリースロットに保存（1234/QWER/ASDF/ZXCV）

※ TASTYは4和音（Maj7/Dom7/Min7）のみ対応。3和音では無効化されます。TASTYとSTOCKは排他動作です。

STOCK Voicing — ピアノボイシング集

鍵盤で良く使われるボイシングを度数表記で収録。ボイシングがはじめから固定されて表示されるので、自分で組む必要がありません。Major / Minor / Dominant / Half-diminished / Diminished / Suspended / Rootless / Spread の全コードタイプに対応。選んだボイシングは自動で64パッドに表示されます。

操作
  • K — STOCK ON/OFF
  • Z / X — 前/次のボイシングへ（TASTY/STOCK共通）
  • ←→ — ボイシング全体を半音トランスポーズ
  • 起動中のボイシングは自動でパッドに表示

Guitar Position — ギターボイシング

ギターのフレットボード上でコードのポジションを自動算出。実際に演奏可能なフィンガリングが表示されます。フレットボードをクリックして手動入力もできます。「パッドに反映」でギターボイシングをパッド配置に変換できます。複数の配置パターンから選べます。

ジャンル別プリセット（HPS会員専用） — Jazz / Bossa / Funk の3ジャンルに最適化されたギターボイシングを表示できます。ジャンルごとにポジショニングの傾向が異なります。

こんな時に使えます

  • 「このコードの押さえ方を調べたい」 → Chord モードでRoot・Quality・Tensionを選択 → ボイシングボックスで配置を確認
  • 「ダイアトニック・コードを覚えたい」 → 5度圏でキーを選択すれば、ダイアトニック・バーに反映されます
  • 「レッスンで出てきたスケールをパッド上で確認したい」 → Scale モードでキーとスケールを選択。31スケール対応
  • 「コード進行を作ってDAWに送りたい」 → コードをメモリースロットに保存 → MIDI Export でDAWにインポート
  • 「このコード、どのキーのダイアトニック？」 → Available Scaleパネルで逆引き。Practical/Diatonicソートで実用性順にも並べ替え可能
  • 「転調先のキーを探したい（ピボットコード）」 → Chord モードで手動コード入力。ダイアトニックバーが消え、Available Scaleがそのコードの属しうる全キーを表示します
  • 「コードを叩いて進行を確認したい」 → メモリースロットにコードを保存して、MIDIコントローラーやPCキーボードで叩いてみてください
  • 「ギターで押さえたコードをパッドで確認したい」 → ギターフレットボードをクリックして入力 → 「パッドに反映」でパッドに配置。複数の配置パターンから選べます
  • 「洗練されたボイシングを知りたい」 → Chord モードでコードを作り T でTASTY（トップノート分類）、K でSTOCK（ピアノボイシング）を起動。度数付きで確認し、メモリーに保存してDAWへ（HPS会員専用機能です）

最初にやること

  1. キーを選ぶ（例: C）
  2. スケールを選ぶ（例: Major）→ パッド上にスケール音が表示される
  3. ダイアトニックバーでコードをクリック（例: Dm7）→ コードトーンが表示される
  4. パッドをクリックして音を鳴らす。ボイシングボックスで配置を変更できます
  5. 気に入ったコードは Option+1234/qwer/asdf/zxcv でメモリーに保存（16スロット対応）

キーボードショートカット

? キーで全ショートカット一覧が見られます。主なショートカット:

コード操作

  • 1〜7 — ダイアトニックコード選択
  • ↑↓ — 転回形切替
  • ←→ — 半音移動（TASTY/STOCK起動中はボイシングごとトランスポーズ）
  • / — テキストコード入力にフォーカス
  • T — TASTY ON/OFF、K — STOCK ON/OFF
  • Z / X — TASTY/STOCKボイシング送り（前/次）
  • O — Omit 5、R — Rootless、S — Shell循環、Shift+D — Drop循環

全般

  • Tab — モード切替（Scale → Chord → Input）
  • Space — 現在のコード/スケールを再生
  • Option+1234/qwer/asdf/zxcv — メモリースロットに保存
  • [ / ] — バンク切替
  • M — Memory/Perform切替

表示トグル（Cmd+Option）

  • ⌘⌥+L Link（弾いた音を全楽器に同期） / G Guitar / B Bass / P Piano / F Circle of Fifths / S Staff
  • ⌘⌥+A Audio / M Memory Slot / T Text Input
  • ⌘⌥+Q Quality/Tension / V Voicing / K Key Display
  • ⌘⌥+H Header + Info Bar

Performモードでは 1234/qwer/asdf/zxcv の4×4グリッドで演奏できます。

ヘッダーアイコン

画面右上に3つのアイコンボタンがあります。

  • 保存 — 現在の設定（音色・エフェクト・表示オプション等）をブラウザに保存します。次回アクセス時に自動復元されます
  • チュートリアル — インタラクティブチュートリアルを一覧表示。各機能の使い方をステップごとに体験できます。アプリ更新時にアイコンが発光してお知らせします
  • ? ヘルプ — ショートカットキー一覧・バージョン履歴・レイアウト設定を表示します。?キーでも開けます

Info Bar — ヘッダー右側に表示されるステータスバー。ボタンやパネルにマウスを重ねると、そのセクションの説明とショートカットキーが表示されます。⌘⌥+Hでヘッダーごと非表示にできます。

カスタマイズ

画面の並び替え（レイアウト）

Chord・Pad・Circleの3つの画面（ペイン）の表示順を変更できます。?ヘルプを開き、下部の「レイアウト」プルダウンから6パターンのプリセットを選べます。設定はブラウザに保存され、次回も同じ並びで表示されます。モバイルではスワイプの順序が変わります。

表示の切り替え

すべての表示セクションを⌘⌥+キーで切り替えられます（Ableton Live式）。

  • Link — ⌘⌥+L。リンクモード。弾いた音がリアルタイムでパッド・ギター・ベース・ピアノに同期表示。楽器が薄暗くなり、弾いた音だけ光ります。複数楽器の対応関係を学ぶのに最適
  • Guitar / Bass / Piano — ⌘⌥+G/B/P。各楽器ダイアグラムの表示切り替え。複数同時表示も可能
  • 五度圏 / 譜面 — ⌘⌥+F/S。排他表示
  • Audio — ⌘⌥+A。サウンドパネルの表示切り替え
  • セクション折りたたみ — ⌘⌥+Q(Quality/Tension) / V(Voicing) / K(Key) / M(Memory Slot) / T(Text Input)。使わないセクションを畳んで画面を整理できます
  • Header + Info Bar — ⌘⌥+H。ヘッダーとInfo Barを一括非表示

♯/♭の切り替え

ルート選択の横にある♯/♭ボタンで異名同音の表記を切り替えられます。例: C# ↔ Db。コード名の表示にも連動します。

サウンドパネル

サウンドパネルでは音色の選択とエフェクトの調整ができます。パネル右の▶ボタンで展開します。

音色プリセット

Rhodes物理モデリング音源（Clean DI）がデフォルトです。サンプルのダウンロードなしで即座に演奏できます。VOICINGノブでピックアップの音色を調整できます。

エフェクト

各ノブをドラッグして値を調整します。

  • DRIVE — アナログサチュレーション。温かみのある歪み
  • Lo Cut — 低域カットフィルター。低音のモコモコを除去
  • Hi Cut — 高域カットフィルター。キラキラしすぎる高音を抑える
  • REVERB — 残響。空間の広がりを加える
  • PHASER — フェイザー。うねりのある揺れ
  • FLANGER — フランジャー。ジェットサウンド的な揺れ
  • TREMOLO — トレモロ。音量の周期的な揺れ

オートフィルター（Auto-Wah）

エンベロープフィルター。FREQ・RESO・SENS・DECAYを調整できます。
